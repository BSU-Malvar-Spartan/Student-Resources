# 🎓 GitHub Student Developer Pack: My Journey & Resources
A guide and repository for documenting my progress as a student developer and the steps I took to unlock the GitHub Student Developer Pack.

---

## 🛠 Step 1: Account Setup & Security
Before applying, I ensured my GitHub profile was professional and secure.
- [ ] **Create GitHub Account:** Signed up at [github.com](https://github.com).
- [ ] **Enable 2FA:** Set up Two-Factor Authentication via **Settings > Password and Authentication**. (Mandatory for the Pack!)
- [ ] **Profile Photo:** Uploaded a clear, professional headshot.
- [ ] **Bio & Name:** Updated my "Public Profile" name to match my Student ID exactly, also add details in account such as location, time zone, and billing address etc..

---

## 📧 Step 2: Academic Verification
The most critical part is linking your identity to your institution.
1. Go to **Settings > Emails**.
2. Add your school-issued email address (e.g., `yourname@university.edu`).
3. **Verify the email:** Click the link sent to your school inbox.
4. Set your school email as your **Secondary Email** (keep your personal as primary if you prefer).

---

## 🚀 Step 3: The Application Process
Here is the workflow I followed to avail the benefits:

### 1. Documentation Prepared
* **Student ID Card:** Must be current (showing the 2025-2026 academic year).
* **Enrollment Letter:** An official PDF from the registrar as a backup.

### 2. Submission
* Navigated to [education.github.com/pack](https://education.github.com/pack).
* Selected "Get Student Benefits."
* **Tip:** if first try got rejected just follow the details required by github, then try again.
